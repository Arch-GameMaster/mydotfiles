## Dependecies

- Rofi
- Polybar
- i3lock

### Needed fonts

- UbuntuSans Nerd Font
- Symbols Nerd Font

## Installation

Move the downloaded `polybar` and `rofi` directories into your `~/.config` directory. And set the `scripts` directory in your `$HOME`
Start `mypolybar.sh` from your terminal and enjoy.
